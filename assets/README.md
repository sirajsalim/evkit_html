# EVKit Screenshot Assets

This folder contains placeholder PNG files for the app screenshot carousel on the homepage.

## File Naming Convention

Each screenshot has two versions: **Dark mode** and **Light mode**

- `evkit_1_dark.png` / `evkit_1_light.png`
- `evkit_2_dark.png` / `evkit_2_light.png`
- `evkit_3_dark.png` / `evkit_3_light.png`
- `evkit_4_dark.png` / `evkit_4_light.png`
- `evkit_5_dark.png` / `evkit_5_light.png`
- `evkit_6_dark.png` / `evkit_6_light.png`

## Image Specifications

- **Aspect Ratio:** 9:19.5 (iPhone aspect ratio)
- **Recommended Size:** 1170 x 2532 pixels (iPhone 14 Pro resolution)
- **Minimum Size:** 900 x 1950 pixels
- **Format:** PNG with transparency
- **File Size:** Keep under 500KB each for optimal web performance

## Screenshot Content Suggestions

1. **Screenshot 1:** Journey cost calculator / Main dashboard
2. **Screenshot 2:** Cost per mile calculation with breakdown
3. **Screenshot 3:** Vehicle comparison (Tesla vs Polestar, etc.)
4. **Screenshot 4:** Battery degradation prediction chart
5. **Screenshot 5:** Savings calculator (EV vs petrol comparison)
6. **Screenshot 6:** Apple Watch / Home Screen widgets

## Design Guidelines

- **Dark Mode:** Black background (#000000), systemMint accents (#62D2A2)
- **Light Mode:** White background (#FFFFFF), systemMint accents (#62D2A2)
- Use iPhone mockup with rounded corners (32px border-radius in CSS)
- Ensure text is readable and UI elements are clearly visible
- Match EVKit's design system (SF Pro typography, minimalist iOS style)

## How to Replace

1. Export your screenshots at the recommended resolution
2. Name them according to the convention above
3. Replace the placeholder files in this folder
4. Images will automatically update on the website (no code changes needed)

## Testing

After replacing images:
1. Load the website and verify all 6 screenshots appear correctly
2. Test the carousel navigation (arrows, keyboard, swipe)
3. Toggle between Dark/Light themes to verify both versions
4. Check responsive display on mobile devices
